3 // Файл INIParser.h - содержит объявление класса INIParser, который является парсером INI файлов.
4 // INI – Файл должен иметь, например такой формат:
5 //
6 // [Section1]
7 // ; Comment
8 // param1 = value ; Comment
9 // param2 = “;vlue;” ; Если точка с запятой используется в значении,
10 // ; нужно чтобы значение было в кавычках
11 // [Section2]
12 // param1 = value ; Имя параметра, должно быть уникальным в пределах
13 // param2 = “;vlue;” ; секции
14 //
15 // Если имя параметра не будет уникальным, то получить значение второго
16 // параметра с одинаковым именем с помощью ф-ии GetValueByName будет невозможно.
17 // Однако его можно получить, перебрав с помощью ф-ии GetSectionLine все пары
18 // имя-параметра/значение для данной секции (это не удобно).
19 //
20 // Чтобы распарсить INI файл нужно использовать объект INIParser
21 // у которого такой интерфейс:
22 
23 // INIParser(const tstring& ini_filename) - конструктор, получает имя ini файла.
24 // Parser() - запускает процесс анализа файла.
25 // SectionList - член класса, предоставляющий интерфейс
26 // к данным ini файла:
27 //
28 // SectionList.size() - возвращает секций в файле
29 // SectionList.empty() - возвращает true, если файл не содержит распознаных секций
30 // SectionList.GetSectionByName(tstring& SectionName) - возвращает указатель на секцию
31 // SectionList.GetSectionByName(..).GetValueByName(tstring& ParamName) - возвращает значение параметра,
32 // для данной секции
33 // SectionList.GetSectionByName(..).SectionName - содержит имя секции
34 // SectionList.GetSectionByName(..).size() - возвращает число параметров в секции
35 // SectionList.GetSectionByName(..).empty() - возвращает true, если секция пуста
36 // SectionList.GetSectionByName(..).GetSectionLine(size_t Num) - возвращает параметр/значение по номеру
